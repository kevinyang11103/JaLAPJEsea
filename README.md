# 图书管理系统 python501

## 项目概述

本系统是一款基于Python 3.6和MS Sqlserver开发的图书管理系统，前端界面采用pyqt5构建。系统主要实现图书的分类管理、读者管理、借阅管理以及系统维护等功能。

## 技术栈

- 后端：Python 3.6
- 数据库：MS Sqlserver
- 前端界面：PyQt5

## 功能模块

### 系统角色

- 管理员
- 普通用户

### 登录注册

- 允许普通用户进行注册。

### 图书查询与借阅查询

- 普通用户可在界面上查询图书信息和个人借阅记录。

### 类别管理

- 对用户类别数据进行增删改查。
- 配置借阅数上限和借阅时间上限。

### 读者管理

- 对用户数据进行增删改查。

### 图书管理

- 对图书数据进行增删改查。

### 系统维护

- 管理员可以修改密码和数据库链接信息配置。

### 借阅管理

- 包括图书借阅、图书归还与账户充值。

### 存储过程

- 图书借阅过程中，系统会判断库存是否充足。
- 判断已借数量是否超过限制。

## 运行环境

- 操作系统：兼容Windows、Linux等操作系统
- Python版本：Python 3.6以上
- 数据库：MS Sqlserver
